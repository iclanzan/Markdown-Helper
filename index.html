Markdown Helper

Built by @jamiebicknell

This is a Javascript based Markdown helper which when you hit the return key, it checks to see if the line is a bullet, numbered list or code block and automatically indents and adds the asterisk, number or spaces. If you hit return again on an empty line, it will remove that item and return to regular paragraph text.

To see it in action, create either a bullet list, numbered list or code block using Markdown syntax and press return to start a new line. To resume regular text input, just press return on an empty line.
